# Wallpaper Calendar Pro v2

Улучшенная версия генератора обоев-календарей для iPhone.

## Что нового
- только iPhone-пресеты, без Android
- расширенный список размеров iPhone
- более удобный мобильный интерфейс
- новые фоновые режимы
- сохранение пользовательских пресетов в localStorage
- номера недель
- разделители кварталов
- badge с числом дней в месяце
- заметка пользователя на обоях
- встроенная инструкция по установке на экран блокировки iPhone

## Запуск
```bash
npm install
npm start
```

Открыть в браузере:
```bash
http://localhost:3000
```

## Основные маршруты
- `/` — интерфейс приложения
- `/wallpaper.svg?...` — SVG-версия обоев
- `/wallpaper.png?...` — PNG-версия обоев
- `/api/options` — список пресетов и тем
